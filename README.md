# Proyecto HTML con estilos - UrbanFit Studio

Proyecto creado para la tarea entregable de CSS.

## Archivos incluidos

- `index.html`
- `css/styles.css`
- `js/script.js`

## Características

- HTML semántico.
- CSS organizado con variables, Flexbox, Grid y responsive design.
- JavaScript para menú móvil y validación básica del formulario.
- Diseño adaptable a móvil, tablet y escritorio.
- Botones, enlaces, formulario y navegación interna.

## Cómo entregarlo

1. Sube todos los archivos a un repositorio de GitHub.
2. Activa GitHub Pages desde `Settings > Pages`.
3. Selecciona la rama `main` y carpeta `/root`.
4. Copia el enlace público y entrégalo en la plataforma.

## Importante

No subas solo el archivo `index.html`. Deben subirse también las carpetas `css` y `js`.
